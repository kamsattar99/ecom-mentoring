# Design Direction: Steep.app / Refero Style — Adapted for Ecom King Brand

## Core Aesthetic
- **Light, warm canvas** — NOT dark theme
- "White canvas with depth" — crisp backgrounds, carefully chosen neutrals, warm accent
- Restrained elegance and confident clarity
- Serif display font for headlines + clean sans-serif for body
- Subtle shadows for gentle elevation
- Generous spacing, breathing room

## Key Design Principles
1. Light backgrounds (#ffffff primary, #f7f7f8 secondary)
2. Warm accent color (#fbe1d1 "Warm Mist") — BUT we adapt this to Ecom King brand colors
3. Serif display headings (large, editorial feel)
4. Clean sans-serif body text
5. 24px border-radius on cards
6. Pill-shaped buttons (very large radius)
7. Subtle shadow stack for depth
8. 80px section gaps
9. Max-width 1280px
10. No heavy saturated colors — muted neutrals

## Ecom King Brand Color Adaptation
- Keep the light canvas (#ffffff) and fog (#f7f7f8)
- Replace "Warm Mist" (#fbe1d1) with a light tint of Ecom King's brand blue
- Keep Ink (#17191c) for primary text
- Use Ecom King's blue as the accent/brand color instead of terracotta
- The user's current brand uses blue tones — we'll create a warm-but-blue accent

## Typography Adaptation
- Signifier (serif) → Use "Instrument Serif" or "Playfair Display" (Google Fonts alternatives)
- Sohne (sans-serif) → Use "Inter" or "DM Sans" (Google Fonts alternatives)
- Actually, to avoid "AI slop" with Inter, let's use "Outfit" or "Satoshi" feel with "DM Sans"

## Layout Patterns from Steep.app
- Full-bleed hero with centered headline over soft gradient background
- Floating UI elements around hero
- Logo bar (Trusted by teams at...)
- Feature sections with left-aligned text + right-side visual
- Tab-based feature showcases
- Customer story/testimonial with large quote + photo
- Grid cards for features (3-4 columns)
- Final CTA section
- Clean footer with column layout

## Components
- Filled Primary Button: Dark bg (#17191c), white text, pill shape
- Ghost Button: Transparent bg, dark border, pill shape
- Default Card: White bg, 24px radius, subtle shadow
- Subtle Card: Fog bg (#f7f7f8), 24px radius, no shadow
- Accent Card: Brand accent bg, 24px radius, no shadow
